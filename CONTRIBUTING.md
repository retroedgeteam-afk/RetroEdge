Contributing to RetroEdge

Thank you for your interest in RetroEdge! 🙌
This project is actively under development and relies on the contributions of the community. Whether you are a developer, designer, tester, or retro gaming enthusiast, you can help shape the future of our consoles and NovaRetroOS.

How to Contribute

Fork and clone the repo

git clone https://github.com/YOUR_ACCOUNT/RetroEdge.git

Create a branch for your changes

git checkout -b my-contribution
Work on your feature, bug fix, or improvement
Follow the coding conventions outlined in the README
For UI/UX design, add mockups and assets to the /assets folder

Commit your changes

git add .
git commit -m "Clear description of your contribution"
Submit a Pull Request (PR)
Describe what you added or fixed
Link your PR to an existing issue if possible
Types of Contributions
Code: NovaRetroOS, drivers, UI, VPS optimization
Design: mockups, icons, animations, HUD
Documentation: README, guides, tutorials
Testing: bug reports, improvement suggestions, UX feedback
Guidelines
Respect all contributors
Be clear in commits and PR descriptions
Test your changes before submitting
Every contribution counts, no matter how small!
Useful Resources
README to understand the project
Roadmap to see priority tasks
“Good first issue” tags for easy starting points

Thank you for contributing to RetroEdge!
Together, we’ll create a modern-retro platform open to everyone. 🎮✨
